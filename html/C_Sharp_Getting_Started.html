1) Within the TeamDynamix waffle, select the downloads icon.
2) On the downloads tab, select the newest version of the TeamDynamix API .DLL file. This file will allow us to use TeamDynamix data types within Visual Studio. Click the "download" button.
3) Open Visual Studio. If you do not have Visual Studio downloaded to your computer, do that now. Then navigate to File->New->Project. Within the "Visual C#" tab, select the "Console App (.NET Core)" project type. You can use other project types if desired, but a Console App will fulfill most of your needs and is simple enough to use.
4) Now, in a seperate File Explorer window, navigate to your Visual Studio project folder. This file path goes as follows: your name->source->repos->project name->project name. Yes, you navigate into two separate, consecutive folders with your project name. Then move the TeamDynamix.Api.dll file you downloaded into this folder.
5) Back in Visual Studio, select Project->Add Reference. We will be adding the .dll file as a project reference.
6) Within the Reference Manager, click "Browse". Navigate to the folder where you just moved the .dll file to. Select the "TeamDynamix.Api.dll" file and click "Add".
7) Hit the file's check mark in the Reference Manager, and click "OK".
8) Add the following line of code to the top of your Program.cs file: "using TeamDynamix.Api.Tickets;" The red squiggle should not appear under it. If the red squiggle does appear, repeat steps 1-6 over again. Test out the file you have included by instantiating a ticket object as seen in the photo.
